Pour développer en local:
------------------------

- pas besoin de wamp, on lance le server http intégré de Symfony :

        "php -S 128.0.0.1:8001 -t public"  à condition que le port 8001 soit libre !

- "git commit -am" avant de quitter le développement

1. développer en local (avec aussi les commandes Symfony en CLI)
2. le mettre dans un Git (git init)
